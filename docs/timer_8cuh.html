Detailed Description

Timer classes for measuring execution time on CPU and CUDA.

This header file contains two classes: Timer and CudaTimer. The Timer class utilizes C++ standard library features to provide high-resolution timing capabilities for CPU processes. The CudaTimer class leverages CUDA events to measure execution time for CUDA kernels efficiently.

Usage:

  • Create an instance of Timer for CPU timing or CudaTimer for CUDA kernel timing.
  • Use the elapsed or elapsed_and_reset methods to retrieve the elapsed time as needed.
  • Make sure to manage CUDA resources properly; the CudaTimer class will automatically clean up upon destruction.
Note
This file should be included in source files that require timing functionalities for performance measurement.
