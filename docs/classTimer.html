Detailed Description

A high-resolution timer class for CPU timing measurements.

This class provides functionalities to measure elapsed time using the high-resolution clock provided by the C++ standard library. It can reset the timer and retrieve the elapsed time since its last reset.

Note
This class is useful for profiling CPU algorithms and measuring performance in time-critical applications.
